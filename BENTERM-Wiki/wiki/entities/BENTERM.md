# BENTERM

BENTERM is the project identity for this WezTerm-based terminal distribution. The repo path for current work is:

```text
<repo-root>
```

## Positioning

- Cross-platform terminal workflow for Windows, Linux, and macOS.
- Built on WezTerm, with separate BENTERM branding and packaging.
- Focused on coding-session quality-of-life: hotkeys, sane defaults, notification behavior, idle glow cues, fresh theme rotation, and visual identity.
- Product idea: BENTERM is WezTerm plus an attention surface for vibe coding.
- The terminal should help users reacquire the right session quickly when several windows are open.
