# Tech Stack

## Core

- Rust workspace managed by Cargo.
- Lua configuration surface via the existing WezTerm config model.
- Native platform integrations for Windows, Linux, and macOS.

## Packaging

- GitHub Actions generated by `ci/generate-workflows.py`.
- Linux package scripts in `ci/deploy.sh`, `ci/appimage.sh`, and Flatpak templates.
- macOS app bundle template under `assets/macos/BENTERM.app`.
- Windows installer and assets under `assets/windows/` and `ci/windows-installer.iss`.
