# BenjaminTerm

BenjaminTerm is a focused fork/rebuild of [[WezTerm]] for a multi-agent coding workflow. It should feel like WezTerm with better identity, better defaults, and stronger agent attention cues.

Current design stance:

- Keep upstream terminal core behavior intact.
- Add distinctive bundled font and theme defaults.
- Improve notification-to-window routing.
- Avoid duplicating [[Hyper Yap]] paste and speech workflows.
