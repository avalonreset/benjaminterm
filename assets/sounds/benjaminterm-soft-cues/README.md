# BenjaminTerm Soft Cues

An 84-file CC0 attention cue set for BenjaminTerm agent-ready notifications.

This pack intentionally replaces the previous `kenney-interface` cue library. No files from the old Kenney Interface Sounds pack are included.

Sources:

- Kenney UI Audio: 51 UI sounds from the `Audio/` directory.
- ObsydianX Interface SFX Pack 1: 33 UI/interface sounds selected from confirm and back tone categories.

Generated format:

- WAV
- Mono
- 44.1 kHz
- 16-bit PCM
- Per-file peak normalized to approximately -14 dB for consistent notification loudness
- Extremely low-energy cues removed after remastering so the shuffle bag does not assign effectively inaudible sounds

See `SOURCE_MANIFEST.json` for source-to-output mapping and `LICENSE_CC0.txt` for source/license notes.
