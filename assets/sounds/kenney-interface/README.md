# Kenney Interface Sounds

Prototype BenjaminTerm agent-attention sounds.

Source: https://opengameart.org/content/interface-sounds

Author: Kenney

License: Creative Commons CC0 1.0 Universal

Original package: `kenney_interfaceSounds.zip`

The source package contains 100 normalized OGG interface sounds. For the
BenjaminTerm prototype, those sounds were converted to mono 44.1 kHz 16-bit WAV
files so Windows can play them through the native `PlaySoundW` API.

Credit to Kenney is appreciated by the author but not required by CC0.
